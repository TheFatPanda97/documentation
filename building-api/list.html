GET buildings/list

Returns a list of buildings.

URL

https://cobalt.qas.im/api/1.0/buildings/list

Parameters

key
required
Your unique API key.

limit
optional
The number of results to return, up to a maximum of 100 per request. The default value is 10.

skip
optional
The number of results to skip. The default value is 0.

Example

https://cobalt.qas.im/api/1.0/buildings/list?limit=2
