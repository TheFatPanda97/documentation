GET courses/show/:id

Returns a specific course identified by the id parameter.

URL

https://cobalt.qas.im/api/1.0/courses/show/:id

Parameters

key
required
Your unique API key.

id
required
The unique identifier for the course.

Example

https://cobalt.qas.im/api/1.0/courses/show/CSC148H1F20159
