Getting started

Cobalt is a collection of open data web APIs that allow you to interface with public information from the University of Toronto.

The goal with Cobalt is to foster a culture of open data at the university. Cobalt's web APIs are free to use and also free to contribute to.

How to start

Cobalt's web APIs are hosted at the following URL: 

https://cobalt.qas.im/api/1.0/

{% if logged_in %} You will need to authenticate your requests with an API key. {% else %} In order to authenticate your requests, you will need to sign up and receive a unique API key. {% endif %} For more information, refer to the authenticating guide.

Suggestions

If you have an idea for a web API that would be a good fit for Cobalt, please let the community know by opening an issue on the GitHub repository.
